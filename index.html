📚 프로젝트 개요

Miracle Reading System은 독서 습관 형성과 도서 관리를 위한 종합적인 웹 애플리케이션입니다. AI 기반 도서 요약, 독서 계획 관리, 속독 훈련, 소셜 기능 등을 제공하는 풀스택 독서 플랫폼입니다.

🎯 구현된 주요 기능

1. 사용자 인증 및 관리 시스템
  • 인증 방식: 폼 기반 로그인 (BCrypt 암호화), Google OAuth2, Kakao OAuth2
  • 회원 관리: 회원 가입, 프로필 관리, 회원 탈퇴 (DeletedUser 테이블 백업)
  • 세션 관리: 동시 접속 제어 (최대 1개 세션), 관리자 전용 세션 분리
2. 도서 관리 시스템
  • 알라딘 Open API 연동: ISBN 기반 도서 정보 자동 수집
  • 일괄 도서 등록: 마크다운 파일 기반 ISBN 리스트 파싱
  • 도서 조회 및 관리: 페이징 처리, AJAX 기반 상세 정보 조회
3. AI 기반 도서 요약 시스템
  • Ollama 로컬 LLM 연동: Qwen3:1.7b 모델 사용, Spring AI 프레임워크 통합
  • 요약 타입: 전체 요약, 간단 요약, 간략 요약, AI 요약
  • 동시 요청 방지: ConcurrentHashMap을 활용한 동시 실행 방지
  • 요약 관리: 키워드 저장 (최대 10개), 질문 저장 (최대 10개), 마인드맵 데이터 저장
4. 독서 계획 및 기록 관리
  • 독서 계획: 주간/월간/년간 목표 설정, 달성률 자동 계산
  • 독서 스케줄 (미션): 도서별 독서 스케줄 생성 및 관리, 상태 관리 (성공/지연 완료/실패/진행중)
  • 독서 기록: 일별 페이지 기록, AJAX 기반 실시간 저장, 누적 페이지 수 자동 계산
  • 독서 통계: 총 요약 개수, 공개/비공개 요약 개수, 평균 요약 길이, 가장 많이 읽은 카테고리
5. 속독 훈련 기능
  • 시각 훈련: 주변 시야 확장 훈련, 동적 시야 훈련, 집중력 향상 훈련
  • 속독 연습: 속도 조절 가능한 텍스트 읽기, 읽기 속도 측정 및 기록
6. 갤러리 및 소셜 기능
  • 도서 요약 갤러리: 공개 요약 조회, 키워드 검색 기능
  • 소셜 인터랙션: 좋아요 기능, 찜(즐겨찾기) 기능, 댓글 시스템 (대댓글 지원)
  • 통계 기능: 인기 요약 통계, 사용자 활동 통계
7. 마인드맵 기능
  • JSON 형식의 마인드맵 데이터 저장 및 조회
  • 도서 요약과 연동
8. 관리자 콘솔
  • 관리자 인증: 별도 세션 관리, BCrypt 암호화
  • 회원 관리: 회원 목록 조회, 등록, 삭제
  • 도서 관리: 도서 등록, 목록 조회, 일괄 등록
